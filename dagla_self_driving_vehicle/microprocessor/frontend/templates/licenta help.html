FAQs

What is this website all about?

The main reason that I created the website is so that I can watch the recording for any problems that can appear on the signal detection algorithm.

What programming languages did you use for this when creating the website?

For backend I used python and javascript and for the frontend part I used HTML and CSS.

What programming languages did you use for the mobile app?

For the mobile app I used for backend java, python and firebase and for the frontend part only XML.

What programming languages did you use for the microcontroller part (Arudino)?

For this only C/C++.

Any difficulties that you encounter while creating this project?

Hmm one in particular... I had some problems when creating the finger detection algorithm because I firstly wanted to do it with a file but the data will not update immediately and so I did it with firebase because of the real time property.

Any weird bugs you had while creating the website?

Yup, weird bugs with threads. I couldn't add a delay in form of sleep in python flask when making a connection between two endpoints so I created my custom one (had problems with this because of the photo when capturing it did not load the frame correcly so I needed to wait 100/150 miliseconds to load).

How much time did you spend on making this project?

I worked on it for like 4 to 5 months with minimum of 4 hours of code in a day.

What was the most fun part in doing this?

The machine learning part I think (the finger detection and signal detection), but also the usage of threads when trying to stop and reload the program in the site.
